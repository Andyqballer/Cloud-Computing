

## Getting Started

First install node.js on your PC https://nodejs.org/en

Second, run the development server:

```bash
npm run dev
# or
yarn dev
# or
pnpm dev
# or
bun dev
```

Third open [http://localhost:3000](http://localhost:3000) with your browser to see the result.
